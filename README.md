# 📧 Responsive Email Template

## 📌 Overview

This project was created as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to design a visually appealing, responsive, and user-friendly email template that delivers important information in a clean, engaging, and professional layout while maintaining readability across different devices.
